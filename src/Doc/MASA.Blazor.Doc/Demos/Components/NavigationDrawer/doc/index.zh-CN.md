`MNavigationDrawer` 是用于导航应用程序的组件。 通常被包装在 `MCard` 元素中使用。

## API

- [MNavigationDrawer](/docs/api/MNavigationDrawer)

## 警告

<!--alert:error-->
如果使用启用了 **App** 属性的 `MNavigationDrawer`，则不需要像示例中那样使用 **Absolute** 属性。
<!--/alert:error-->
